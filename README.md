# estoque 

Este sistema ajuda as pessoas a organizarem um estoque, deixando o trabalho e as empresas mais rápidos e funcionais.Como o menu funciona: Na sua tela vão aparecer algumas opções. 
Cada uma tem seu próprio número e sua respectiva função.
Digite 1 para permitir ao usuário inserir os dados de um novo produto no estoque
Digite 2 para solicitar um ID e exibir os dados do produto correspondente, se ele existir
Digite 3 para mostrar a lista de produtos disponíveis
Digite 4 para permitir ao usuário alterar a quantidade de um produto existente
Digite 5 para sair do estoque
